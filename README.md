# Smart Pantry Intelligente Lebensmittelverwaltung
Smart Pantry ist eine geplante Desktop-Anwendung zur Verwaltung von Lebensmitteln im Haushalt. Das Programm soll Bestände und Haltbarkeiten erfassen, Rezept- und Wochenvorschläge erstellen und automatisch Einkaufslisten generieren. Ziel ist es, Lebensmittelverschwendung zu reduzieren und die Essensplanung zu erleichtern.
